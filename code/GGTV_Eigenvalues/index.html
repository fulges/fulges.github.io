Supplementary Files for

Characteristic polynomials and eigenvalues of tensors

by F. Galuppi, F. Gesmundo, E. Turatti, L. Venturello

We collect the supplementary material required to complete the computer calculations described in Section 6 of the paper. The calculations are performed in Macaulay2.

Tangent Cones to the Discriminant of Plane Cubics

We collect the files used in Brysiewicz-Gesmundo-Steiner for the computation of the tangent cones to the discriminant of plane cubics.

  • The file cubicInvs.m2 contains a hard-coded expression of the discriminant of plane cubics.
  • The file tangentConesPlaneCubics computes the tangent cone at each normalized singular cubic.

Characteristic Polynomial of Plane Cubics

We collect the files required for the computational part in the proof of Theorem 6.1. The computation was performed on a Lenovo ThinkBook 14 G2 ITL, running Debian 11, on Macaulay2 v. 1.17.1. All results were returned almost instantaneously except for the first subcase of the case where the plane curve defined by f is a line and a double line, which required a few minutes.

  • The file cubicInvs.m2 contains a hard-coded expression of the discriminant of plane cubics.
  • The file planeCubicsH.m2 performes the calculation explained in the proof of the Theorem.
